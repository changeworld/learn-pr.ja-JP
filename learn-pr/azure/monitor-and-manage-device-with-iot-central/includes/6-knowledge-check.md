Azure IoT Central を使用すると、IoT ソリューションの接続、監視、および管理が簡単になります。 このモジュールでは、マシンにメンテナンスの問題が発生したときに監視や管理を行うために、リモートのコーヒー マシンを Azure IoT Central に接続するというエンドツーエンドのシナリオを作成しました。
